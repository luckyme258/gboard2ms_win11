# 微软拼音词库管理器 使用手册

## 一、简介

微软拼音词库管理器是一个图形化工具，用于管理微软拼音输入法的自定义短语词库。支持从 Gboard 导入词库、导出为 Gboard 格式、导出为微软拼音 DAT 格式。

## 二、安装与运行

### 方式一：Python 脚本运行
```bash
python ms_manager.py
```

### 方式二：直接运行 exe
双击 `微软拼音词库管理器.exe` 即可运行。

**注意**：`book.txt` 必须与 exe 放在同一目录下。

## 四、基本操作

### 4.1 搜索
在搜索框中输入文字，实时过滤显示匹配的词条（匹配短语或编码）。

### 4.2 添加词条
1. 点击「添加」按钮
2. 填写短语（如：`你好`）
3. 填写编码（如：`nh`）
4. 选择候选位置（1-9，数字越小越靠前）
5. 点击「确定」

### 4.3 编辑词条
- **双击**表格中的任意词条
- 或**右键**点击 → 选择「编辑」
- 修改后点击「确定」

### 4.4 删除词条
- **选中词条**（支持多选），按 `Delete` 键
- 或点击「删除」按钮
- 或右键 →「删除」

### 4.5 多选技巧
| 操作           | 效果              |
| -------------- | ----------------- |
| `Ctrl + 点击`  | 选中/取消选中单个 |
| `Shift + 点击` | 连续选中多个      |
| `Ctrl + A`     | 全选              |
| `Delete`       | 删除选中          |

## 五、导入导出

### 5.1 导入 Gboard 词库
1. 在手机上打开 Gboard 设置 → 词典 → 个人词典 → 导出
2. 将导出的 `dictionary.txt` 传到电脑
3. 点击「导入 Gboard」，选择该文件
4. 程序自动去重合并

### 5.2 导入微软拼音 DAT
1. 在微软拼音设置中导出用户自定义短语（得到 `.dat` 文件）
2. 点击「导入 DAT」，选择该文件
3. 程序自动解析并合并

### 5.3 导出 Gboard 格式
1. 点击「导出 Gboard」
2. 选择保存位置
3. 生成的 `.txt` 文件可直接导入手机 Gboard
4. **语言标签自动判断**：包含中文 → `zh-CN`，纯英文 → `zz`

### 5.4 导出微软拼音 DAT
1. 点击「导出 DAT」
2. 选择保存位置
3. 生成的 `.dat` 文件可在微软拼音中导入使用

### 5.5 保存
点击「保存」将当前数据保存到 `book.txt`（程序会自动备份为 `book.txt.bak`）

## 八、常见问题

### Q1: 打开程序提示「book.txt 不存在」
A: 程序会自动创建，首次运行正常现象。

### Q2: 导入 Gboard 文件提示「编码错误」
A: 确保 Gboard 导出的文件是 UTF-8 编码。

### Q3: 导出的 DAT 导入微软拼音后乱码
A: 检查 `book.txt` 中的短语是否正确，重新导出。

### Q4: 如何备份词库？
A: 直接复制 `book.txt` 文件即可。

### Q5: 支持哪些 Windows 版本？
A: Windows 10 / Windows 11。
